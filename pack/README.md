# RISC-V UnifiedDB Support in VS Code

This is an extension pack containing:

* Syntax highlighting for IDL files
* Syntax highlighting for AsciiDoc with embedded Ruby
* Syntax highlighting for C++ with embedded Ruby

## Features

* Syntax highlighting for IDL files
* Syntax highlighting for AsciiDoc with embedded Ruby
* Syntax highlighting for C++ with embedded Ruby
